Probando postgresql en google

Apr 15, 2017   #google  #cloud  #postgresql 

Dando de alta la base de datos

Pues basta con hacer un nuevo proyecto en la Consola, poner nuestra tarjeta de credito, y despues arrancar una instancia de postgresql 9.6 una vez arrancada, te dice la ip que tiene, y te puedes conectar a ella como siempre:

 pgcli -h IP_GOOGLE -U postgres BASE_DATOS_RECIEN_CREADA

# Poniendo postgis Para nuestra app necesitamos guardar un raster con el modelo digital del terreno, es decir con las altitudes de cada pixel. En lugar de guardarlo en un fichero GeoTiff u otro formato, lo guardaremos en una tabla postgis con tipo raster.

Google cloud platform viene con postgresql 9.6 y postgis 2.3 disponible, asi que podremos hacer uso de esta tecnologia:

 $ CREATE EXTENSION postgis;

y ya tenemos postgis en la nube de google. xd
